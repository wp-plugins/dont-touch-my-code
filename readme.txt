== Description ==

Adds [code] shortcode that saves your code from the WordPress WYSIWYG editor

== Installation ==

1. Upload `dont-touch-my-code/` directory to the `/wp-content/plugins/`
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[code] Your code [/code]` to your page content
